# Java Variable

ใน Python เมื่อเราต้องการให้ตัวแปรใด ๆ แทนค่าใด ๆ เราจะให้ `a = สิ่งที่เราต้องการ` ได้เลย

แต่ใน Java เราจำเป็นต้องระบุด้วยว่า a ที่เราต้องการจะให้แทนนั้น มีชนิดเป็นอะไร ซึ่งจะคล้ายๆลักษณะต่อไปนี้

`<ประเภทตัวแปร> <ชื่อตัวแปร> = สิ่งที่เราต้องการ`

เช่น

`int x = 72;`
`float y = -44.32f;`
`String name = "Cat is CUTE!";`

โดยประเภทตัวแปรหลักๆมีดังนี้

## ตัวเลข

ตัวเลขในภาษา Java หลักๆจะมีสองชนิดคือ จำนวนเต็มและไม่เต็ม

### Integer(จำนวนเต็ม)

จำนวนเต็มในภาษา Java นั้นมีหลายแบบให้คุณได้เลือกคัดสรร ซึ่งแต่ละแบบจะเก็บได้ไม่เท่ากัน

- `byte` เก็บได้ตั้งแต่ `-128 ถึง 127` (2^16)
- `short` เก็บได้ตั้งแต่ `-32768 ถึง 32767` (2^16)
- `int` เก็บได้ตั้งแต่ `-2147483648 ถึง 2147483647` (2^32) **(นิยม)**
- `long` เก็บได้ตั้งแต่ `-9223372036854775808 ถึง 9223372036854775807` (2^64)

> หากจะใส่ตัวเลขในตัวแปร long ให้ พิมพ์ตัวอักษร L ติดท้าย เช่น 1250L

#### :thunder_cloud_rain: วิชามาร (อาจทำให้คุณปวดหัวได้)

หากเขียนนำหน้าด้วย 0 จะกลายเป็นการเขียนเลขฐาน 8 เช่น

```java
int int_a = 045;
```

`int_a` จะกลายเป็น 37 ไม่ใช่ 45

หากเขียนนำหน้าด้วย 0x หรือ 0X จะกลายเป็นการเขียนเลขฐาน 16 เช่น

```java
int int_a = 0x45;
```

`int_a` จะกลายเป็น 69 ไม่ใช่ 45


### Float(จำนวนไม่เต็ม หรือ จำนวนจริง)

ในภาษา Java ก็จะมีจำนวนเต็มอยู่สองชนิด

- `float` - ใช้ในการเก็บค่าทศนิยม ซึ่งสามารถเก็บได้มากที่สุด `+- 3.4028235E38`
- `double` ใช้ในการเก็บค่าทศนิยม ซึ่งสามารถเก็บได้มากที่สุด `+- 1.7E308`
>ทำนองเดียวกับภาษา python ทั้ง float และ double จะเก็บค่าประมาณเท่านั้น อาจไม่ได้ค่าจริงแปปเป๊ะๆ

> โดยทั่วไป การเขียนตัวเลขทศนิยม จะถูกตีความเป็นตัวแปรประเภท double โดยอัตโนมัติ หากอยากให้เป็นประเภท float ให้ใส่ f ต่อท้ายตัวเลข เช่น 52.3f


### การดำเนินการของตัวเลข

การดำเนินการของตัวเลขจะเหมือนคณิตศาสตร์ปกติเลข

```java
int int_a = 7;
int int_b = 3;

float float_c = 4.5f;
float float_d = 1.5f;
```

- `+` ... มันก็คือ... บวก
- `-` ... มันก็คือ... ลบ
- `*` ... มันก็คือ... คูณ

ความสำคัญอยู่ตรงนี้
- `/` ... มันก็คือ... หาร
แต่ถ้านำ **จำนวนเต็ม** หารด้วย **จำนวนเต็ม** ผลลัพธ์ก็จะได้จำนวนเต็มโดยการ**ปัดเศษทิ้ง**

```java
int result = int_a/int_b;
System.out.println("result = " + result);
```
> result = 2

ส่วนจำนวนจริงก็ไม่มีปัญหา :)

- `%` คือ Modulo หรือ หารเอาเศษ

สังเกตุว่า ไม่มีสัญลักษณ์ `ยกกำลัง`
> ว้ายๆๆๆๆๆ

จริงยังมีตัวดำเนินการอีก แต่เราจะไม่พูดถึงในคราสนี้

### Shortcut Operators

ในหลายครั้งเราอาจจะชอบนับเลขทีละตัวสองตัว ซึ่งเราสามารถเขียนแบบนี้ได้

```java
int count = 0
count = count + 2;//count = 2
count = count + 2;//count = 4
count = count + 3;//count = 7
count = count + 3;//count = 10
count = count + 3;//count = 13
```

สังเกตุว่า มันจะทำการแทนตัวเองและทำการบวกเพิ่ม ซึ่งทำงานได้ แต่มันอาจจะลำบากและยาวจำเกินไป Java จึงสามารถย่อได้เป็นดังนี้

`+=` `-=` `*=` `/=` `%=`

เช่น `x += 7` มีความหมายเดียวกับ `x = x + 7`

และสุดท้าย บางครั้งเวลาเรานับทีละ 1 เราสามารถเขียนแบบ `x += 1` ได้ ในภาษา Java มีสัญลักษณ์พิเศษคือ `x++`  ซึ่งมีความหมายเหมือนกับ `x += 1` และ `x = x + 1`

นอกจาก `x++` ก็ยังมี `++x` `x--` และ `--x` 

#### :thunder_cloud_rain: วิชามาร (อาจทำให้คุณปวดหัวได้)

ความแตกต่างระหว่าง `x++` กับ `++x`

ถ้าเราเขียนโง่ๆ ระหว่าง `x++` กับ `++x` จะไม่ต่างกัน แต่ถ้ามี**~~ใบตอง~~ตัวแปร**มารองรับ จะมีความต่างกัน

```java
int magic = 7;
int xpp = magic++;

magic = 7;//Reset
int ppx = ++magic;
System.out.println("magic++ is " + xpp)
System.out.println("++magic is " + ppx)
```
> magic++ is 7

> ++magic is 8

จะเห็นว่า `xpp` จะมีค่าเป็น 7 และ `ppx` จะมีค่าเป็น 8 นั้นเพราะว่า `magic++` มันคือการ **ส่งค่าก่อนค่อยบวก** แต่ `++magic` คือการ **บวกก่อนค่อยส่งค่า**

## อักขระ (char)

- char ใช้ในการเก็บตัว**อักษร 1 ตัว**

เช่น 

```java
char alphabet = 'A';
```

สังเกตุว่า ตัวอักษรจะใช้ ' '(Single quote)

ตัวแปร char ในร่างที่เรารู้จักจะทำการเก็บเป็นตัวอักษร แต่ร่างซาตานของมันคือ **จำนวนเต็มประเภท short ที่ไม่มีติดลบ**

เพราะงั้น การดำเนินการของ char จะเหมือนการดำเนินการตัวเลข[**จำนวนเต็มทุกประการ**](https://github.com/CoE30-KKU/EN811302/blob/master/Java101/Chapter%201%20Variable.md#%E0%B8%81%E0%B8%B2%E0%B8%A3%E0%B8%94%E0%B8%B3%E0%B9%80%E0%B8%99%E0%B8%B4%E0%B8%99%E0%B8%81%E0%B8%B2%E0%B8%A3%E0%B8%82%E0%B8%AD%E0%B8%87%E0%B8%95%E0%B8%B1%E0%B8%A7%E0%B9%80%E0%B8%A5%E0%B8%82)

เช่น
```java
char c = (char) ('A' + 'B');
```
c จะไม่ใช่ 'AB' เพราะ char คือการเก็บตัว**อักษร 1 ตัว** เพราะฉะนั้น c ก็จะเป็นตัวอักษรลึกลับตัวหนึ่ง

> แล้วจะรู้ได้ไง

จากที่บอกไปว่า ร่างซาตานของมันคือ **จำนวนเต็มประเภท short ที่ไม่มีติดลบ** แล้วถ้าอยากรู้ว่า ตัวเลข 65 เป็นตัวอักษรอะไร สามารถดูได้ที่ [รหัส ASCII](https://www.cs.cmu.edu/~pattis/15-1XX/common/handouts/ascii.html) แต่ปัจจุบันน่าจะใช้รหัส Unicode เอา ที่สามารถรับภาษาอื่นๆได้

ตัวอย่างการดำเนินการ

```java
char c = 'A';
c += 3;
System.out.println("c is " + c);
c -= 2;
System.out.println("c is " + c);
c++;
System.out.println("c is " + c);
c *= 54;
System.out.println("c is " + c);
```
```
c is D
c is B
c is C
c is ย
```

จริงๆตัวอักษรหลายๆตัวอักษร(สายอักขระ) เราจะเรียนว่า String แต่ขอข้ามก่อนน้าาา

## ตรรกะ (boolean)

- boolean หรือเครื่องหมายตรรกศาสตร์ ใช้ในการเก็บค่าระหว่าง `จริง (true)` และ `เท็จ (false)` *สังเกตว่า true กับ false ทั้งหมดเป็น**ตัวพิมพ์เล็ก***

เช่น
```java
boolean x = true;
```
### การสร้าง boolean
นอกจากที่เราจะกำหนดให้เป็น `true` กับ `false` เราก็สามารถกำหนดผ่านเงื่อนไขหรือการเทียบค่าได้ เช่น

```java
int x = 3;
boolean someBool = (x == 3);
boolean someBool2 = (x < 3);
```
กรณีนี้ `someBool` จะมีค่าเป็น `true`
กรณีนี้ `someBool2` จะมีค่าเป็น `false`

ซึ่งเครื่องหมายเปรี่ยบเทียบมีดังนี้

- `==` (เท่ากับ 2 ตัว) ก็คือการเทียบเท่ากัน
- `!=` ก็คือการเทียบไม่เท่ากัน
- `<` ก็คือการเทียบน้อยกว่า
- `>` ก็คือการเทียบมากกว่า
- `<=` ก็คือการเทียบน้อยกว่าหรือเท่ากับ
- `>=` ก็คือการเทียบมากกว่าหรือเท่ากับ

ตัวอย่าง

```java
int a = 2;
int b = 5;
int c = 7;

boolean b1 = (a == 2);//true
boolean b2 = (a <= 2);//true
boolean b3 = (a > 2);//false
boolean b4 = (a == b);//true
boolean b5 = (a <= b);//true
boolean b6 = (a > c);//false
boolean b7 = (a != c);//true
boolean b8 = (a == b);//false
```

### การดำเนินการของตรรกะศาสตร์

มีไม่เยอะมาก

- `&&` มันก็คือ `และ`
- `||` มันก็คือ `หรือ`
- `!` มันก็คือ `นิเสธ` หรือก็คือตรงข้าม

| P     | Q     | P && Q | P \|\| Q | !P    |
|-------|-------|--------|----------|-------|
| true  | true  | true   | true     | false |
| false | true  | false  | true     | true  |
| true  | false | false  | true     | false |
| false | false | false  | false    | true  |


## สายอักขระ (String)

จริงๆ String ไม่ใช่ประเภทตัวแปรแรกเริ่มในภาษา Java สังเกตว่า S เป็นตัวใหญ่ ดังนั้น String จึงเป็น Class คลาสนึง แต่เพราะว่าเราใช้(โคตรบ่อย) จึงขอจัดมันเป็นตัวแปรอีกประเภทหนึ่งละกัน :)

- String ใช้ในการเก็บข้อความ (เก็บได้หลายตัว)

เช่น

```java
String someStr = "Hewwo";
```

>ภาษา Java การใช้ ' '(Single quote) และ " "(Double quote) จะถือว่าเป็นคนละความหมายกัน! โดย 'A' หมายถึงอักขระ(char)  และ "CoE" หมายถึงสายอักขระ(String)

### การดำเนินการของอักขระ

การดำเนินการมีแค่ตัวเดียว

- `+` (บวก) การบวกของ String คือการนำ String มาต่อกัน เช่น

```java
String someStr = "Hewwo";
String someStr2 = "world";

String mixStr = someStr + someStr2;//Hewwoworld
```

แต่การ `+` (บวก) ของ String นั้น ***ท ร ง พ ลั ง ม า ก  ก ก ก ก ก ก ก***

เพราะสามารถบวกตัวแปรได้ทุกๆประเภท


```java
String someStr = "Hewwo";
int num = 45;
char chr = '!';
float ff = 62.7f;
boolean bl = false;

String mixStr = someStr + num + chr + ff + bl;//Hewwo45!62.7false
```

### การให้บริการของอักขระ

จริงการให้บริการของอักขระมีเยอะมากๆๆๆๆๆๆๆๆๆๆๆ ดังนั้นจะขอแสดงแค่หลักๆที่ใช้บ่อยๆ

```java
String magicStr = "MiracleYEaH!!";
```

#### `magicStr.length()`
คือการนับว่า String นั้นยาวกี่ตัว

```java
int len = magicStr.length()
System.out.println(len)
```
> 13

#### `magicStr.equals(AnotherString)`
คือการตรวจสอบว่า String มีข้อความเดียวกับ AnotherString หรือไม่ (คล้ายๆ `magicStr == AnotherString`) เช่น

```java
if (magicStr.equals("MiracleYEaH!!")){
    ...
}
```

#### `magicStr.toUpperCase()`
#### `magicStr.toLowerCase()`
คือการส่งข้อมูลให้่ทุกๆตัวเป็นตัวพิมพ์ใหญ่หมด หรือพิมพ์เล็กหมด เช่น

```java
String upp = magicStr.toUpperCase();
String low = magicStr.toUpperCase();
System.out.println(upp);
System.out.println(low);
System.out.println(magicStr);
```

```
MIRACLEYEAH!!
miracleyeah!!
MiracleYEaH!!
```
> สังเกตุว่า magicStr ก็ไม่เปลี่ยนแปลงจากเดิม


#### `magicStr.charAt(int index)`
คือการดูว่าตัวอักษรที่ `index` คือตัวอะไร

```java
char a = magicStr.charAt(0);
char b = magicStr.charAt(1);
char c = magicStr.charAt(5);
System.out.println(a);
System.out.println(b);
System.out.println(c);
```

```
M
i
l
```

#### `magicStr.substring(int begin[,int end])`
คือการตัดเฉื่อน String จากตัวที่ `begin` จนถึงก่อน `end`

หากไม่ใส่ `end` จะทำการแทนถึงตัวสุดท้าย

```java
String x = magicStr.substring(2);
String y = magicStr.substring(4,6);

System.out.println(x);
System.out.println(y);
```

```
racleYEaH!!
cl
```
